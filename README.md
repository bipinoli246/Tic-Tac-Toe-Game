# ❌⭕ Tic Tac Toe

A simple Python project that lets two players play Tic Tac Toe in the console.  
It provides a user-friendly interface to place moves and automatically determines the winner or a tie.

---

##  Features
- Two-player game  
- Check for winning or tie conditions automatically  
- User-friendly console interface

##  Tech Used
- Python
